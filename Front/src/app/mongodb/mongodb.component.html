Es un sistema de base de datos no relacional o NoSQL creado en C++ orientado a documentos. Es un software de licencia libre y funcional en diferentes sistemas operativos.
El formato en el que almacena la información MongoDB es BSON o Binary JSON. Además, a consola de busqueda (MongoDB Shell) es hecha sobre Javascript, lo que quiere decir que se hacen las consultas sobre este lenguaje.
