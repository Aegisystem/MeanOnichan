Es un sistema de base de datos no relacional o NoSQL creado en C++ orientado a documentos. Es un software de licencia libre y funcional en diferentes sistemas operativos.
El formato en el que almacena la información MongoDB es BSON o Binary JSON. Además, a consola de busqueda (MongoDB Shell) es hecha sobre Javascript, lo que quiere decir que se hacen las consultas sobre este lenguaje.

Historia de MongoDB

  • Escalabilidad

    problemas de escalabilidad con los sistemas de bases de datos tradicionales.

  • 10gen

    Dwight Merriman, Eliot Horowitz y Kevin Ryan fundan 10gen para construir una PaaS.

  • Lanzamiento

    • Febrero: MongoDB 1.0 con funciones básicas.
    • Diciembre: MongoDB 1.2 con procesamiento de datos a gran escala.
    • Servicios de soporte y capacitación.
    • Reconocimiento de la marca.

  • Versiones 1.4 y 1.6

    • Marzo: MongoDB 1.4 con creación de índices en segundo plano
    • Agosto: MongoDB 1.6 con fragmentación para producción con escalabilidad horizontal

  • Crecimiento

    • Aumento de empleados y oficinas con soporte 24/7
    • MongoDB 2.2 con ueva colección de productos de suscripción

  • Reforzando

    • La organización cambia su nombre 10gen a MongoDB Inc.
    • 250 empleados y 1000 clientes
    • Marzo: MongoDB 2.4 con el motor V8 de Google.
    • Adquisición de WiredTiger

  • Almacenamiento

    • Marzo: MongoDB 3.0 con nuevo motor de almacenamiento
    • MongoDB versio 3.2
